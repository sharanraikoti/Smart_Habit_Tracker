# Smart Habit Tracker

A Smart Habit Tracker built using Python and Streamlit that helps users track daily habits, monitor productivity, build consistency, and analyze their progress through interactive dashboards and reports.

## Features

- ✅ Daily Habit Tracking
- 📊 Interactive Dashboard
- 📈 Habit Performance Analysis
- 🏆 Best & Worst Habit Detection
- 🔥 Habit Streak Tracking
- 📅 Weekly & Monthly Reports
- ⚠️ Risk Detection
- 📂 Category Analysis
- 🎯 Achievement System
- 📤 CSV Export
- 🔄 Reset Data

## Technologies Used

- Python
- Streamlit
- Matplotlib
- File Handling

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/Smart-Habit-Tracker.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run streamlit_app.py
```
